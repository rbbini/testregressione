------------------------- Progetto Epimetheus -------------------------

Questo progetto mira allo sviluppo di una piattaforma che fornisca
supporto decisionale ai medici attraverso delle visualizzazioni.

In particolare i medici potranno vedere, attraverso una serie di
grafici, le predizioni fatte da alcuni modelli di machine learning,
dati in ingresso una serie di dati sul paziente.

Inoltre sarà possibile cambiare il valore di alcuni dati di input
direttamente nella pagina dei risultati, in modo da vedere il variare
delle predizioni al variare delle features.


Per utilizzare la piattaforma bisogna installare i pacchetti necessari
per il funzionamento del back-end (si può fare sia da cmd che con il 
supporto di un IDE). 
Si consiglia di installare i pacchetti in un ambiente nuovo.


Nel file "Docs.txt" è possibile leggere una breve descrizione delle 
funzioni presenti nel back-end, mentre nel file "TODO.txt" sono scritte
delle possibili aggiunte/modifiche da apportare al progetto.
